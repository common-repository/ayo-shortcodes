Yet another shortcodes plugin for WordPress.

== Description ==

This plugin add visual design elements by using WordPress shortcode API.
Take a look at our [free Genesis child theme](http://demo.ayothemes.com/ayoshop/shortcodes/) for demo and documentation.

== Installation ==

1. Upload the entire 'AyoShortcodes' folder to the '/wp-content/plugins/' directory
2. DO NOT change the name of the 'ayo-shortcodes' folder
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Use any available shortcodes at WP Editor
